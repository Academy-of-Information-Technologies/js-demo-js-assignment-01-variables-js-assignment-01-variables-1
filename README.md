# Assignment 1 — Variables Basics

### Instructions

შეასრულე ქვემოთ მოცემული ნაბიჯები:

1.  შექმენი ცვლადი `firstName` და მიუთითე შენი სახელი.
2.  შექმენი `lastName` და მიუთითე გვარი.
3.  შექმენი `fullName` სადაც იქნება firstName + space + lastName.
4.  შექმენი `age` და მიუთითე ნებისმიერი რიცხვი.
5.  შექმენი `city` — დაამატე ქალაქი.
6.  შექმენი `output` ტექსტი ფორმატით:
    "My name is [fullName]. I am [age] years old and I live in [city]."

        ⚠️ არ გამოიყენო ფუნქციები (function).

    კოდი წერე `src/index.js` ფაილში.

როცა დაასრულებ, commit/push გააკეთე.  
Autograding ტესტები ავტომატურად დაგიფიქსირებენ ქულებს.
